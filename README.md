Laundry Service Website

This is a simple and responsive landing page for a laundry service. It is built using HTML and CSS, focusing on clean layout, basic styling, and smooth user interactions.

Features
Responsive design for desktop, tablet, and mobile
Navigation bar with logo and links
Hero section with heading, description, and image
Interactive call-to-action button with hover effects
Technologies Used
HTML5
CSS3 (Flexbox and media queries)
How to Use

Download or clone the project, then open the index.html file in any browser to view the website.

What I Learned
How to structure a webpage using HTML
Using Flexbox for layout and alignment
Making websites responsive with media queries
Creating hover effects using CSS transform and transitions
Using pseudo-elements for animations
Mistakes and Improvements
Missed proper class selectors (like forgetting . in .cta-btn)
Minor syntax errors such as missing semicolons
Faced layout issues on smaller screens initially
Improved CSS organization and readability
Learned the importance of testing responsiveness
Purpose

This project was created to practice front-end development concepts such as layout design, responsiveness, and CSS animations.
